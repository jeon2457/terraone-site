1️⃣ Cloudflare에서 여러 사이트/도메인 관리

Cloudflare 계정 하나로 여러 도메인을 등록하고 관리할 수 있습니다.

각 도메인마다 별도의 네임서버(NS) 변경이 필요합니다.

즉, 한 계정으로 여러 사이트를 관리 가능하지만, 도메인별로 Cloudflare 네임서버를 적용해야 합니다.

2️⃣ 도메인별 적용 방식

Cloudflare 계정에 로그인 → “도메인 추가(Add Site)” 선택

도메인 이름 입력 → Cloudflare가 제공하는 네임서버(NS) 주소 확인

도메인 등록처(Domain Registrar, 예: dothome)에서 기존 네임서버 → Cloudflare 네임서버로 변경

변경 후 최대 24~48시간 정도 전파(Propagation) 필요

참고: Cloudflare는 각 사이트마다 SSL, CDN, 방화벽, 캐시를 별도로 설정할 수 있습니다.
